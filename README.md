# MLIR Learning

A structured exploration of MLIR from first principles.

## Goals

- Learn MLIR by understanding its design decisions.
- Connect compiler engineering with formal methods.
- Build small working examples.
- Design a custom dialect.
- Explore verification and semantics.

## Repository Structure

- `notes/` — Learning notes
- `blog/` — Draft blog and LinkedIn posts
- `examples/` — Small MLIR programs
- `experiments/` — Passes, dialects, and prototypes
- `design-decisions.md` — Design insights and tradeoffs
